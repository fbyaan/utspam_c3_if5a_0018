Aplikasi Flutter ini merupakan proyek UTS (Ujian Tengah Semester) untuk mata kuliah Pemrograman Aplikasi Mobile. Aplikasi ini menampilkan berbagai komponen UI Flutter dalam bentuk yang terstruktur dan menarik.

## Fitur Utama
Registrasi & Login: Validasi input dan penyimpanan akun ke database lokal.
Beranda (Home): Menampilkan rekomendasi mobil, pencarian interaktif, dan navigasi modern.
Daftar Mobil: Katalog lengkap mobil yang tersedia untuk disewa.
Formulir Sewa: Input data penyewaan dengan fitur Date Picker dan kalkulasi harga otomatis.
Riwayat Transaksi: Melihat status pesanan (Aktif/Dibatalkan) dan total biaya.
Edit & Batal: Fitur untuk mengubah jadwal sewa atau membatalkan pesanan.
Profil Pengguna: Menampilkan informasi akun yang sedang login.
